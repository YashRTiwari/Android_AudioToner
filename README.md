# Audio Toner
This app generates sound, using MediaPlayer of Android.

Functionality:

<ul>
  
  <li>Sine, Square and Triangular Wave Generation</li>
  <li>Frequency Range - 0 Hz to 20kHz </li>
  <li>Amplitude: 1- 100</li>
  
</ul>
